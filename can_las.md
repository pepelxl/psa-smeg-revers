### ID 0x305  
**Источник: датчик угла поворота руля, длина 6, частота 10 мс**  
Пример: 022B0007F300  
D0[7]-D1[0] положения руля. 0 - руль прямо.  
D2 ускорение руля  
D3[7] направление движения  
D4 Контрольная сумма и инкремент  
```
byte checksumm_305(const byte* frame)
{
    static byte iter = 0;
    byte cursumm = 0;
    cursumm = frame[0] ^ frame[1] ^ frame[2] ^ frame[3];
    cursumm = ((cursumm >> 4) ^ (cursumm & 0x0F)) ^ iter;
    cursumm = (cursumm << 4) ^ iter;
    iter = ++iter & 0x0F; 
    return cursumm;
}
```
D5 повторяет значения D2

### ID 0x452  
**Источник BSM, длина 5, частота 50 мс**  
Транслирует заданные режимы для радара от BSI  
Пример: 0000000200  
D0[5] сигнал левого поворотника  
D0[4] сигнал правого поворотника  
D3[3] запрос активации функции предупреждения о столкновении(ARC). Появляется при наборе скорости выше 30км/ч, если пользователем выбран пункт меню.  
D3[1-0] чувствительность срабатывания функции предупреждения о столкновении.  
&nbsp; 1 - близко  
&nbsp; 2 - норма  
&nbsp; 3 - далеко  
D4[1] запрос активации функции экстренного торможения(FARC). Появляется при наборе скорости выше 20км/ч, если пользователем выбран пункт меню.

### ID 0x4F6  
**Источник: радар, длина 5, частота 100 мс**  
Транстлирует состаяние объекта для круиз-контроля   
Пример: FFFE5FFE00 объект отсутствует  
Пример: 1B054DEA10 2,7 секунды, 5 метров, объект удаляется  
Пример: FE034DAC10 машина стоит на светофоре с зафиксированным объектом  
D0 время в секундах до объекта. Value / 10.  
D1 растояние до объекта в метрах.  
D2-D3 относительная скорость объекта. Единицы измерения не известны. Предположительно завязано на частоту радара и/или доплеровский сдвиг. Иследуемая деталь имеет значение 19884 единицы при котором скорости объекта и радара равны. Не факт, что другой радар будет иметь тоже значение из-за кодировки или калибровки. Значения выше говорят о том, что объект удаляется. Значения ниже говорят о том, что объект приблежается. Так же есть вероятность, что старшие четыре бита используются для других задач  
D4
